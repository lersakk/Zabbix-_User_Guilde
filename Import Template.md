## Import Template steps


__Step 1 : ดาวน์โหลด Template ที่ต้องการใช้มาไว้ที่เครื่อง__


__Step 2 : ไปที่ zabbix__

1.) ไปที่ Data collection >> Template >> Import

2.)กด choose file

3.)เลือก Template ที่ต้องการใช้ ที่ดาวน์โหลดไว้
